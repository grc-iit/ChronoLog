# Code of Conduct

ChronoLog follows the [Contributor Covenant v2.0](https://www.contributor-covenant.org/version/2/0/code_of_conduct/). The full text is in [`.github/CODE_OF_CONDUCT.md`](https://github.com/grc-iit/ChronoLog/blob/develop/.github/CODE_OF_CONDUCT.md).

## Key Points

- **Be respectful** — treat all community members with courtesy regardless of background or experience level.
- **No harassment** — personal attacks, trolling, derogatory comments, and unwelcome sexual attention are not tolerated.
- **Scope** — the code of conduct applies in all project spaces (GitHub issues, PRs, discussions) and when representing the project publicly.
- **Enforcement** — violations are handled through a graduated response: correction, warning, temporary ban, or permanent ban, depending on severity.

## Reporting

If you experience or witness unacceptable behavior, contact the project team at **scslab@iit.edu**. All reports are reviewed confidentially.
